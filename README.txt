HOW TO RUN 5-Minute Multiboxing:
1) Unzip contents into your classic dir
2) Edit toonlist.txt  ... put in your info. See toonlist for instructions.
3) double click 5mmb_Classic.bat. You can close the resulting window.
4) Double click Hotkeynet.exe
5) Load your 5mmb_HKN_classic.txt
6) scroll lock on
5) ctrl-alt-m to login
**************
NOTE: The first time you login, it will ask you what region you are in. close all windows but one. Select region. Hit okay. When you get to the login screen, close it. Then do ctrl-alt-M again.
**************
6) when you are in, close any popups. ctrl-i to initialize all your wow buttons
8) 0 key in your main window to invite
9) accept invites to the left and clockwise aournd the screen in that order
10) Interface Options--turn all action bars on. Make sure click-to-move is on.
11) alt-4 to make everyone follow
12) 2 to attack a single target. F-twice to make all toons attack/talk target.
13) F1-F5 heals main window then next window clockwise
14) alt-3 heals tank.
